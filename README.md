# Winterfell Cousins Launcher

Launcher customizado para o modpack Winterfell Cousins (Minecraft 1.21 + Fabric).

[![Download](https://img.shields.io/badge/⬇_BAIXAR_LAUNCHER-v1.4-green?style=for-the-badge)](https://github.com/gustavo-oliveira-rosa/winterfell-cousins-launcher/releases/download/v1.4/minecraft-launcher-1.0.jar)

## Como usar

1. Baixe o `.jar` clicando no botão acima
2. Dê duplo clique no arquivo
3. Insira seu nickname e clique em **JOGAR**

## Requisitos

- Java 21+ instalado
- 4GB+ de RAM
- Linux (Ubuntu/Debian) ou Windows 10/11
