# Winterfell Cousins Launcher

Launcher customizado para o modpack Winterfell Cousins (Minecraft 1.21 + Fabric).

[![Download](https://img.shields.io/badge/⬇_BAIXAR_LAUNCHER-v1.0-green?style=for-the-badge)](https://github.com/gustavo-oliveira-rosa/winterfell-cousins-launcher/releases/download/v1.0/minecraft-launcher-1.0.jar)

## Como usar

1. Baixe o `.jar` clicando no botão acima
2. Execute com: `java -jar minecraft-launcher-1.0.jar`
3. Insira seu nickname e clique em **JOGAR**

## Requisitos

- Java 25 (o launcher oferece instalação automática)
- 4GB+ de RAM
- Linux (Ubuntu/Debian)
