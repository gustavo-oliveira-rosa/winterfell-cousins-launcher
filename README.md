## Como usar

1. Baixe o `.jar` e o script do seu sistema (`.sh` para Linux ou `.bat` para Windows)
2. Coloque ambos na mesma pasta
3. Dê duplo clique no script (`.sh` ou `.bat`)
4. Insira seu nickname e clique em **JOGAR**

> **Linux:** Se o `.sh` não abrir com duplo clique, clique com botão direito → Propriedades → Permissões → marque "Permitir execução como programa"

## Requisitos

- Java 21+ instalado
- 4GB+ de RAM
- Linux (Ubuntu/Debian) ou Windows 10/11
