# Winterfell Cousins Launcher

Launcher customizado para o modpack Winterfell Cousins (Minecraft 1.21 + Fabric).

[![Download](https://img.shields.io/badge/⬇_BAIXAR_LAUNCHER-v1.1-green?style=for-the-badge)](https://github.com/gustavo-oliveira-rosa/winterfell-cousins-launcher/releases/download/v1.1/winterfell-launcher.sh)
[![Download JAR](https://img.shields.io/badge/⬇_BAIXAR_JAR-v1.1-blue?style=for-the-badge)](https://github.com/gustavo-oliveira-rosa/winterfell-cousins-launcher/releases/download/v1.1/minecraft-launcher-1.0.jar)

## Como usar

1. Baixe os dois arquivos (`.sh` e `.jar`) e coloque na mesma pasta
2. Dê duplo clique no `winterfell-launcher.sh`
3. Insira seu nickname e clique em **JOGAR**

> Se o `.sh` não abrir com duplo clique, clique com botão direito → Propriedades → Permissões → marque "Permitir execução como programa"

## Requisitos

- Java 25 (o launcher oferece instalação automática)
- 4GB+ de RAM
- Linux (Ubuntu/Debian)
